Data types specify different sizes and values that can be stored in a variable.

The primitive data types include boolean, char, byte, short, int, long, float, and double.
The non-primitive data types include Classes, Interfaces, and Arrays.

The standard data types are:
•	Integer(int) - 4 byte
•	Floating numbers(float) - 4 byte
•	Character(char) - 1 byte
•	Boolean(boolean) - 1 byte
•	Double(double) - 8 byte
•	Short(short) - 2 byte
•	Long(long) - 8 byte

Example:
```
public class DataTypesTutorial {
    public static void main(String[] args) {

        byte x = 5;
        int age = 10;
        short height = 34;
        float rateOfInterest = 7.1f;
        double rate = 89.65498165;
        boolean isThis_SeriesCool = false;
        char myCharecter = 'A';

        System.out.println(isThis_SeriesCool);
    }

}
